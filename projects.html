Bastion Auto Splitter

[C#]

Video game speedrunners want to know when they've reached a certain point in the game so they can compare their splits against previous runs. With the help of DevilSquirrel, we built this LiveSplit plugin for Bastion players to automatically time when they have completed each level.

BastionBot

[Java, Spring]

BastionBot is a Discord bot that automatically tells the Bastion Speedrun Community when players have gone live on Twitch and when new speedruns have been verified on Speedrun.com. It was recently rewritten to use the Spring Framework.
